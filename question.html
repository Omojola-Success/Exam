Tags used in HTML

  1. /href/ : It serves as a link

  2. /div/ : It helps to arrange an html in sections

  3. /h1 - h6/ : It changes text font size and this defines html headings

  4. /br/ : It's used to break line statement

  5. /html/ : defines the root of an html document

  6. /body/ : defines the document's heading

  7. /abbr/ : It defines an abbreviation or an acronym

  8. /blackquote/ : It specifies a quoted section

  9. /head/ : This contains the metadata

  10. /meta/ : defines the metadata, it is usually inside the /head/

  11. /header/ : For introductory content

  12. /hr/ : This defines a line break in html

  13. /img/: It is uses to introduce/display an image on the html page

  14. /pre/ : Used to preformat a text

  15. /title/ : This defines the document's name

  16. /style/ : It is used to define style information (css) for a document

  17. /table/ : This is used to define a table

  18. /tr/ : Used to defines the beginning of a row in a table

  19. /td/: Used to define the beginning of a cell

  20. /u/ : Used to underline a text


Different types of list in HTML

  1. /ol/ : This is an ordered list. It is used when you want to list statements using numbers i.e 1,2,3,4..... It is used to group a set of related items in a specific order

  2. /ul/ : This is an unordered list. It is used when you want to list statements using dots(.) It is used to group a set of related items in no particular order.

  3. /dl/ : This is a definition list. It is used to list statements in a defined way. when using the /dd/ tag, the statement shifts to the right but when using the /dt/ tag, the statement remains at it's normal pattern. It is used to display name/value pairs such as terms and definitions.


CSS

CSS can be used to add beauty to the html. HTML is just like the skeleton while CSS is the flesh. CSS is used to style and layout web pages - for example, to make change to the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features. Without CSS, websites would look very boring. There are many reasons why you will want to use CSS in web design. First, CSS can save you time. Second, CSS is efficient. Lastly, it's easy for users to learn and update, which makes global changes to style simple and quick.



Host domain

An host domain service sells domains and registers them to an owner in exchange for a yearly subscription fee. A domain host, such as example.com, is an internet service that handles your domain name by which visitors can use to find one. A domain is set up like this; www.YOURWEBSITENAME.com. The usual domain names are .com, .net, .org, .edu, .edu. By buying a domain from a domain host, you gain ownership of that domain for as long as you pay the yearly fee. DNS records are used by domain hosts to connect your domain name to email, websites, and other web services. To mention a few, popular domain hosts include GoDaddy, enom, DreamHost, and 1&1 IONOS. Hosting services work by maintaining stable and secure storage spaces. While domain host provide more than just simple data storage, it's a core part of their functionality. Hosts store data on hardware which allows for easy maintenance and access by online users.
